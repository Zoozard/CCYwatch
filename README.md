# 📈 CCYwatch

Веб-сервис на базе Django для автоматического мониторинга, глубокого анализа и визуализации исторических данных курсов валют на основе официального API Центрального Банка Российской Федерации.

## 🛠️ Используемый стек технологий
* **Backend:** Python 3.12, Django 6.0.6
* **Аналитика данных:** Pandas 3.0.3, NumPy 2.4.6 (расчёт волатильности, средних, минимальных и максимальных значений за период)
* **Парсинг данных:** lxml 6.1.1, Requests 2.34.2 (отказоустойчивый парсинг XML ЦБ РФ с обходом шлюзов безопасности)
* **Frontend:** Bootstrap 5.3 (адаптивная вёрстка), Chart.js 4.4 (интерактивные графики трендов)
* **База данных:** SQLite (индексируемые таблицы с защитой от дублирования записей)

## 🌟 Ключевой функционал
1. **Автоматический парсинг ЦБ РФ:** При посещении главной страницы система проверяет актуальность данных и фоново докачивает свежий XML-пакет, обновляя базу данных.
2. **Глубокая аналитика (Pandas):** Мгновенный расчёт ключевых финансовых показателей валюты за исторический период.
3. **Интерактивные графики:** Построение динамических линий тренда без создания статических картинок на сервере.
4. **Личный кабинет & Авторизация:** Разграничение прав доступа (Гость / Авторизованный пользователь).
5. **Внутренние уведомления (Watchlist):** Возможность установить целевой триггер стоимости валюты. При достижении условием истинности, пользователю на экран выводится всплывающее уведомление.

## 📦 Быстрый запуск проекта

1. **Клонируйте репозиторий и перейдите в папку:**
   ```bash
   cd CCYwatch
   ```
2. **Активируйте виртуальное окружение:**
   ```bash
   venv\Scripts\activate
   ```
3. **Установите все зависимости:**
   ```bash
   pip install -r requirements.txt
   ```
4. **Запустите локальный веб-сервер:**
   ```bash
   python manage.py runserver
   ```
5. **Откройте проект в браузере:** `http://127.0.0`
