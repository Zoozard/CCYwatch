# Техническое задание: Сервис мониторинга и аналитики курсов валют (CCYwatch)

## 1. Цель проекта
Создание отказоустойчивого веб-сервиса для автоматического отслеживания курсов валют ЦБ РФ. Система предназначена для сбора исторических данных, проведения финансового анализа за выбранный период и настройки пользовательских уведомлений при достижении валютой целевой стоимости.

## 2. Стек технологий и архитектура
* **Backend:** Python 3.12, Django 6.0.6
* **Парсинг и API:** lxml 6.1.1, Requests 2.34.2 (оптимизированные сессионные запросы к xml_daily Центробанка РФ)
* **Аналитика данных:** Pandas 3.0.3, NumPy 2.4.6 (проведение математических и статистических расчетов)
* **Frontend:** Bootstrap 5.3 (интерфейс), Chart.js 4.4 (интерактивная визуализация трендов)
* **База данных:** SQLite (с использованием индексов уникальности для защиты от дублирования записей)
* **Статические файлы:** Полностью локализованы в директории проекта для автономной работы без внешних CDN-зависимостей.

## 3. Роли пользователей
* **Гость:** просмотр таблицы актуальных курсов валют, доступ к аналитическим показателям и интерактивным графикам трендов.
* **Авторизованный пользователь:** регистрация/вход в систему, формирование личного списка отслеживания (Watchlist), установка индивидуальных триггеров на целевую стоимость валюты.

## 4. Модели данных

### Currency (Валюта)
* `name` (CharField) — полное название валюты
* `char_code` (CharField, unique) — буквенный код (например, USD)
* `num_code` (CharField) — цифровой код
* `nominal` (IntegerField) — номинал валюты

### ExchangeRate (История курсов)
* `currency` (ForeignKey) — связь с моделью `Currency`
* `rate` (DecimalField) — курс валюты к рублю
* `date_checked` (DateField) — дата фиксации курса ЦБ РФ
* *Ограничение:* `unique_together = ('currency', 'date_checked')`

### Watchlist (Список отслеживания)
* `user` (ForeignKey) — связь со встроенной моделью `User` Django
* `currency` (ForeignKey) — связь с моделью `Currency`
* `target_rate` (DecimalField) — целевой курс
* `is_notified` (BooleanField) — статус срабатывания триггера

## 5. Ключевой функционал (User Stories)
* **Автоматическое обновление:** При загрузке главной страницы система проверяет базу данных на текущую дату и, при отсутствии записей, безопасно скачивает свежий XML-пакет с сайта ЦБ РФ.
* **Аналитика Pandas:** На детальной странице валюты фреймворк Pandas мгновенно рассчитывает минимум, максимум, средний курс и волатильность за весь доступный в базе исторический период.
* **Интерактивные графики:** На основе массивов данных из Pandas библиотека Chart.js строит динамическую линию тренда с отображением точных котировок при наведении.
* **Личный кабинет & Валидация:** Авторизованные пользователи могут выставить целевой курс (с защитой от ввода отрицательных чисел).
* **Внутренние уведомления (UX):** Если курс валюты опускается ниже установленной цели, пользователю при входе на сайт выводится зеленая всплывающая плашка Django Messages. Сработавший алерт автоматически скрывается из интерфейса аналитики, но сохраняется в базе данных для истории.
