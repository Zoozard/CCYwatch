# Техническое задание на проект: CBR Currency Observer

## 1. Цель проекта
Создание веб-сервиса для автоматического отслеживания курсов валют ЦБ РФ. Система помогает пользователям анализировать исторические данные изменений курсов, строить графики.

## 2. Роли пользователей
* **Гость:** просмотр текущих курсов валют, доступ к аналитическим графикам за 30 дней.
* **Пользователь:** формирование личного списка отслеживаемых валют.

## 3. Модели данных

### Currency (Валюта)
* **name** — название валюты
* **char_code** — буквенный код (например, USD)
* **num_code** — цифровой код
* **nominal** — номинал

### ExchangeRate (История курсов)
* **currency** — внешний ключ на модель `Currency`
* **rate** — курс валюты к рублю
* **date_checked** — дата фиксации курса

### Watchlist (Список отслеживания)
* **user** — внешний ключ на стандартную модель `User`
* **currency** — внешний ключ на модель `Currency`
* **target_rate** — целевой курс

## 4. Ключевой функционал (User Stories)
* **Просмотр котировок:** Пользователь видит таблицу актуальных курсов валют с расчетом изменения по сравнению с предыдущим днем.
* **Визуализация трендов:** На странице конкретной валюты отображается интерактивный график (Plotly или Matplotlib) с историей изменения курса за выбранный период.
* **Автоматическое обновление:** Система по расписанию (или принудительно через админку/скрипт) загружает свежие XML-данные с сайта ЦБ РФ и обновляет базу данных.

## 5. Внешние интеграции и аналитика
* **Интеграция:** Получение ежедневных данных в формате XML через официальное API Центробанка РФ ([cbr.ru/development/SXML/](https://www.cbr.ru/development/SXML/)).
* **Аналитика:** Использование библиотеки Pandas для расчета волатильности, средних, минимальных и максимальных значений курсов за месяц. Отрисовка графиков с помощью Matplotlib или Plotly.
